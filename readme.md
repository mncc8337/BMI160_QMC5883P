# BMI160_QMC5883P
An extension library providing QMC5883P magnetometer support for the BMI160 via its secondary I2C bus. 

**Hardware Requirements:** You must have a QMC5883P magnetometer wired directly to the BMI160's auxiliary I2C interface to use this wrapper. For detailed BMI160 wiring instructions, please refer to the base [BMI160-Arduino repository](https://github.com/mncc8337/BMI160-Arduino/tree/master).
